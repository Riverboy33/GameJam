# Clicker Pygame (style Cookie Clicker)

## Prérequis
- Python 3.10+

## Installation
```bash
pip install -r requirements.txt
```

## Lancer le jeu
```bash
python main.py
```

## Contrôles
- Clique sur le grand bouton (le “cookie”) pour gagner des pièces.
- Achète des bâtiments dans la boutique pour générer des pièces automatiquement (CPS = coins per second).
- Achète l'upgrade de clic pour augmenter les coins par clic.
- `S` : sauvegarder
- `R` : reset (efface la sauvegarde)
- `ESC` : quitter

## Sauvegarde
Le jeu sauvegarde automatiquement dans `save.json` (dans le même dossier) et tu peux aussi forcer la sauvegarde avec `S`.
